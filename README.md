# Locmais Andaimes

Site dinamico para uma empresa de construcao que loca andaimes. A aplicacao usa
Node.js, Express e EJS para renderizar as paginas no servidor e salvar videos e
opinioes de clientes em `data/content.json`.

## Recursos

- Visual em tons de bordo e cinza.
- Pagina inicial com apresentacao da empresa e servicos.
- Cadastro de videos por link do YouTube.
- Blog/depoimentos com postagens de clientes.
- Dados persistidos no backend em arquivo JSON.
- Layout responsivo para desktop e celular.

## Como rodar

```bash
npm install
npm start
```

Depois acesse `http://localhost:3000`.

Para desenvolvimento com reload automatico do Node:

```bash
npm run dev
```
